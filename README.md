# Nicole Lai-Lopez Portfolio

This is the personal portfolio website for Nicole Lai-Lopez, built with modern HTML5, CSS3, and a touch of JavaScript. It is inspired by the clean, professional style of [skysheng7.github.io](https://skysheng7.github.io/).

## Features
- Fully responsive, clean, and modern design
- All content from Nicole's latest CV
- Sections: About, Education, Projects, Research, Publications, Industry, Awards, Skills, Interests, Contact
- Easy to update and extend

## Structure
- `index.html` — Main HTML file
- `css/style.css` — Main stylesheet
- `js/script.js` — Placeholder for interactivity
- `images/` — Place your profile picture as `profile.jpg` and favicon as `favicon.ico`

## Deployment
1. Add your profile picture to `images/profile.jpg` (recommended size: 400x400px, square, professional headshot).
2. Add a favicon to `images/favicon.ico` (optional).
3. Commit and push to your GitHub repository.
4. Enable GitHub Pages in your repository settings (root or `/docs` folder).
5. Visit your portfolio at `https://<your-github-username>.github.io/<repo-name>/`.
