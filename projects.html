fwi-predict, Python Package Contributor | Fish Welfare Initiative 2025.03

Developed fwi-predict, an end-to-end forecasting pipeline that monitors water quality in Eastern India's aquaculture ponds, reducing the need for costly manual sampling in remote areas. The pipeline predicts pond acidity levels using weather forecasts and geospatial data. The system integrates Earth Engine-based data export, cloud bucket downloads, time-aware feature engineering, and a custom multiclass calibration curve visualizer.
